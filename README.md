# **Thyroid Cancer Recurrence Prediction**

##**Project Overview**
This project focuses on predicting the probability of recurrence in patients who have had thyroid cancer and are currently in remission. Using Machine Learning models, we analyze clinical data to identify high-risk patterns.

### **Key Features:**
- **Comparative Study:** Analyzed and compared 5 different Machine Learning models.
- **Performance Metrics:** Evaluated models using Accuracy, Precision, Recall, F1-Score and ROC-AUC to ensure clinical relevance.
- **Data Processing:** Specialized handling of medical datasets, including cleaning and feature engineering.

### **Technologies Used:**
- Python (Pandas, NumPy, Scikit-learn)
- **Data Visualization:** Matplotlib & Seabor
- Jupyter Notebooks

### **Documentation**
The complete analysis, code, and visualizations are available in the .ipynb file in this repository. (Technical documentation is provided in Spanish).
